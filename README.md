Community Complaints Web with **VueJS** and also **CodeIgniter4** as the main backend, interested in trying it?

## Features
* Admin
  * Add Officer
  * Verify Report
  * Validate Report
  * Display Officer List
  * Display Society List
  * Display Report
  * Remove Officer & Society
* Logout
* Officer
  * Verify Report
  * Validate Report
  * Display Report
  * Delete Account
* Profile
* Publics
  * Make Report
  * Display Report
  * Delete Account
* SignIn
